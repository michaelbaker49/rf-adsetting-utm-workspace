## Overview

RF Adsetting Workspace is a web-first utility for building adsets and generating UTM parameters. Instead of splitting those tasks across multiple tools, it keeps them together in one workspace so campaign assets and tracking links can be prepared more efficiently.

The project is aimed at people who handle recurring marketing launches and need a practical way to organize structured adset details. With the workflow centered on setup and parameter creation, it helps reduce repetitive work and makes campaign links simpler to keep in order.

---

## What it does

- Builds adsets for campaign setup
- Creates UTM parameters for tracking links
- Runs in the browser with no desktop installation needed
- Uses a workspace layout to group creation tasks together
- Fits marketing, growth, and traffic coordination workflows
- Supports cleaner campaign naming and link parameter organization
- Keeps the interface focused on the core creation process
